DEPURAR EL CODIGO JAVASCRIPT

CÓDIGO MAL ESCRITO

El tag script permite añadir código JS. Con la consola del inspector de código del Navegador depuramos los errores.
El error: "documen.wri is not a function" indica que "documen.wri" no es un nombre de función reconocible por JS. Seguramente esté mal escrito.

ORDEN DEL JS

Si el código JS accede a un elemento HTML, es importate que al ejecutar el JS ya se haya cargado el HTML. El atributo defer en el tag script asegura que se ejecute el JS después de cargar el HTML

CASE SENSITIVE
