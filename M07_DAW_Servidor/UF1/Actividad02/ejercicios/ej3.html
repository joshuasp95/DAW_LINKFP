Ejercicio 3

Realizar una aplicación que simule una calculadora. Constará de una página HTML con dos campos de texto para introducir los dos operadores, y 4 RadioButtons para seleccionar la operación: suma, resta, multiplicación y división. Se mostrará el resultado de la operación en una segunda página con la frase “El resultado de la suma/resta/multiplicación/división es XXX”. Hacer el programa usando un formulario de tipo POST.
